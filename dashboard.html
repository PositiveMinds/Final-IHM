Import Data

Import patient and appointment data from CSV files

Quick Import

Select a CSV file to import patient or appointment data to your facility.

Import Requirements
Patient Data Requirements
  • CSV format with headers
  • Required: full_name, age, gender, phone, email, primary_condition
  • Optional: region, facility
  • Maximum 1000 records per import
Appointment Data Requirements
  • CSV format with headers
  • Required: patient_id, full_name, next_appointment, appointment_time, status
  • Optional: primary_condition, notes, facility
  • Date format: DD/MM/YYYY

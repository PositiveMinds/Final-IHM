Import Patients Data

Upload a CSV file to import multiple patient records. Maximum file size: 5MB

Required columns: first_name, last_name, email, phone_number, date_of_birth, gender, national_id
Preview Records (First 5)
First Name Last Name Email Phone DOB
CSV Format Guide

Your CSV file should have these columns:

  • first_name - Patient's first name
  • last_name - Patient's last name
  • email - Patient's email address
  • phone_number - Contact number
  • date_of_birth - DOB (YYYY-MM-DD)
  • gender - M, F, or O
  • national_id - ID number

Example CSV format:

first_name,last_name,email,phone_number,date_of_birth,gender,national_id
John,Doe,john@example.com,0123456789,1990-01-15,M,12345678
Jane,Smith,jane@example.com,0987654321,1985-05-20,F,87654321
Tip: Keep your CSV file clean with no extra spaces or special characters in column names.
Import Appointments Data

Upload a CSV file to import multiple appointment records. Maximum file size: 5MB

Required columns: patient_id, appointment_date, appointment_time, appointment_type, status
Preview Records (First 5)
Patient ID Date Time Type Status
CSV Format Guide

Your CSV file should have these columns:

  • patient_id - Patient's unique ID
  • appointment_date - Date (YYYY-MM-DD)
  • appointment_time - Time (HH:MM)
  • appointment_type - Regular, Follow-up, Lab, Consultation
  • status - Scheduled, Completed, or Cancelled

Example CSV format:

patient_id,appointment_date,appointment_time,appointment_type,status
1,2024-01-25,09:00,Regular,Scheduled
2,2024-01-25,10:30,Follow-up,Scheduled
3,2024-01-25,14:00,Lab,Completed
Tip: Ensure patient IDs exist in your system before importing appointments.

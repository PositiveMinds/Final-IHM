Mobile View: Resize your browser to less than 768px width and click the hamburger menu to test the sidebar sliding animation.

Test Instructions

  1. On desktop (768px+): Sidebar should always be visible on the left
  2. On mobile (<768px):
    • Sidebar should be hidden initially (off-screen to the left)
    • Click the hamburger button to toggle the sidebar
    • The sidebar should slide in smoothly from the left
    • A dark overlay should appear behind the sidebar
    • Click the overlay or the hamburger button again to close the sidebar

CSS Animation Details

Sidebar transition: left 0.35s cubic-bezier(0.4, 0, 0.2, 1)

Overlay transition: opacity 0.35s cubic-bezier(0.4, 0, 0.2, 1), visibility 0.35s

Mobile breakpoint: 767px

Sidebar width: 250px

Initial position: left: -250px

Active position: left: 0

Debugging

Open browser DevTools (F12) and check:

  • Console: Look for "toggleSidebar" function logs
  • Elements: Check if `.active` class is being added/removed from sidebar and overlay
  • Styles: Verify that the sidebar's `left` property changes from `-250px` to `0`
